# Custom / Blank — Role Preset

No role-specific defaults. `/brief-setup` will ask you for every field from scratch.

## Integration defaults
- calendar: on (required)
- gmail: on (required)
- zoom: on (required)
- jira: ask
- hubspot: ask

## Recommended poll interval
60 min (default). Adjust based on your workflow's cadence.

## What work offers look like (any role)
- Update / append a Confluence page
- Add a comment to a Jira ticket
- Draft a Slack reply (never auto-sent)
- Add / update / mark-done a task in your tasks.md
- Add a read-only note to a HubSpot deal

## Blocked by default
- Sending email
- Changing Jira status / assignee / priority
- Changing HubSpot deal stage / amount / owner / close date
- Deleting anything

To request new allowed actions, file an issue.
